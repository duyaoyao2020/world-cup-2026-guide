# 2026 世界杯动态观赛指南

一个使用 React、Vite、TypeScript 与 React Three Fiber 制作的世界杯动态观赛网页。

## 功能

- 104场世界杯静态赛程与北京时间/当地时间展示
- A-L组积分榜，根据已录入赛果自动计算
- 多场比赛的双队3D阵型与可点击球员资料
- 桌面端半场悬停扩展、球员信息卡及手机端响应式布局
- 本地透明球员头像与无头像降级展示

## 本地运行

```bash
npm install
npm run dev -- --host 127.0.0.1 --port 4173
```

访问 `http://127.0.0.1:4173`。

## 验证

```bash
npm test -- --run
npm run build
```

## 数据说明

- 赛程和赛果是截至北京时间 **2026年6月13日** 的静态快照，不接入实时比赛API。
- 所有阵容均为交互演示阵容，不代表官方实时首发。
- 积分表根据项目内已录入比分自动计算。
- 球员头像素材保存在本地 `头像/` 目录；使用与再分发时应自行确认相应素材授权。
